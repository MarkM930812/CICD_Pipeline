# Frontend

React + Vite dashboard for the spacecraft mission control demo, including fleet switching and a telemetry/history log for each spacecraft.

## Available commands

```powershell
npm install
npm run dev
npm run lint
npm run build
```

During local development, `/api` requests are proxied to the backend at `http://localhost:8080`.

See the repository root `README.md` for the full stack overview and API description.
